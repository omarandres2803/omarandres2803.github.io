Top 50 Cryptocurrencies

This react web application uses the CoinGecko API to list information about the current top 50 cryptocurrencies. Information like: Price (in USD), the growth of the coin in the last 24h, the marketcap and the circulating supply.
It also allows the user to search for a coin, and filters using the name of the cryptocurrency or its symbol, as shown below:

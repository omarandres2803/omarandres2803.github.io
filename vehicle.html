VEHICLE TELEMETRY SYSTEM

In this project the principal goal was to give the client the possibility to track his car's position on a map, while he could also see some other features as: speed , engine revolutions per minute, tires pressure and more, in real time.
To collect all this information, we used an OBDII connected directly to the car, which was accessed via Bluetooth with the driver's cell phone.
Additional to this, the client could also see his car's data records searching by date, not only on a web page but also in a mobile application.
